Get FuncUnit

There are two ways to get Funcunit:

  • Download it
  • Git it from GitHub

We prefer you use Git. It's a great way to share changes with us and the community.

GitHub Repositiory

FuncUnit is a free, open source, web application testing framework that focuses on making tests fun and enjoyable.

If you know jQuery, you already know the API.
